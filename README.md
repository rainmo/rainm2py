    + 递归函数：在函数内部，可以调用其他函数。如果一个函数在内部调用自身本身，这个函数就是递归函数
      阶乘n! = 1 x 2 x 3 x ... x n，用函数fact(n)表示: fact(n)可以表示为n x fact(n-1)
      '''
      def fact(n):
        if n==1:
          return 1
        return n * fact(n - 1)
      '''
      + 理论上，所有的递归函数都可以写成循环的方式，但循环的逻辑不如递归清晰
